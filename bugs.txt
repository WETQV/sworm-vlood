**2. Player.gd — применять статы из GameManager**

В `_ready()` добавь применение выбранного класса:

```gdscript
func _ready() -> void:
    add_to_group("player")
    _apply_class_stats()
    # ... остальной код

func _apply_class_stats() -> void:
    var data = GameManager.CLASS_DATA[GameManager.selected_class]
    var stats = data["stats"]
    
    speed = float(stats["speed"])
    attack_damage = stats["damage"]
    health_component.max_health = stats["hp"]
    health_component.current_health = stats["hp"]
    body_sprite.color = data["color"]
```

**3. Уникальная механика для каждого класса**

Самое интересное — чем классы отличаются в бою. Вот простые идеи:

**Маг** — вместо удара мечом кидает снаряд. В `_perform_attack()` проверяй класс:

```gdscript
func _perform_attack() -> void:
    if GameManager.selected_class == GameManager.PlayerClass.MAGE:
        _shoot_projectile()
        return
    # обычная атака мечом...

func _shoot_projectile() -> void:
    var bullet = preload("res://scenes/projectile.tscn").instantiate()
    bullet.global_position = global_position
    bullet.direction = (get_global_mouse_position() - global_position).normalized()
    bullet.damage = attack_damage
    get_parent().add_child(bullet)
```

**Разбойник** — рывок при атаке:

```gdscript
func _perform_attack() -> void:
    if GameManager.selected_class == GameManager.PlayerClass.ROGUE:
        var dash_dir = (get_global_mouse_position() - global_position).normalized()
        _knockback_velocity = -dash_dir * 400.0  # рывок вперёд
    # дальше обычная атака...
```
