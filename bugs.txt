Проблема в строке 16 файла `player.tscn`. Смотри на эту часть:

```
[node name="CollisionShape2D" type="CollisionShape2D" parent="."]
shape = RectangleShape2D.new()
size = Vector2(32, 32)
```

**Главная ошибка** — `RectangleShape2D.new()` и `size` так не задаются в `.tscn` файлах. Форма должна быть объявлена как отдельный ресурс, а не создаваться через `.new()`. У тебя таких мест несколько в файле.

Исправь `.tscn` вот так — добавь ресурсы форм в заголовок:

```
[gd_scene load_steps=9 format=3 uid="uid://player_scene"]

[ext_resource type="Script" path="res://scripts/components/health_component.gd" id="1_health"]
[ext_resource type="Script" path="res://scripts/components/hurtbox_component.gd" id="2_hurtbox"]
[ext_resource type="Script" path="res://scripts/components/hitbox_component.gd" id="3_hitbox"]
[ext_resource type="Script" path="res://scripts/player/player.gd" id="4_player"]

[sub_resource type="RectangleShape2D" id="1"]
size = Vector2(32, 32)

[sub_resource type="RectangleShape2D" id="2"]
size = Vector2(32, 32)

[sub_resource type="RectangleShape2D" id="3"]
size = Vector2(40, 40)
```

И замени все `shape = RectangleShape2D.new()` на ссылки:

```
[node name="CollisionShape2D" type="CollisionShape2D" parent="."]
shape = SubResource("1")

[node name="CollisionShape2D" type="CollisionShape2D" parent="Hurtbox"]
shape = SubResource("2")

[node name="CollisionShape2D" type="CollisionShape2D" parent="Attack"]
shape = SubResource("3")
position = Vector2(30, 0)
```

**Важно** — обнови `load_steps=5` на `load_steps=9` (4 внешних + 3 формы + 2 запаса), иначе Godot тоже выдаст ошибку.

Самый простой способ избежать таких проблем в будущем — **не редактировать `.tscn` файлы вручную**, а создавать сцены через редактор Godot. Он сам генерирует правильный формат.
